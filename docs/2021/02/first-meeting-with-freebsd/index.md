# 与FreeBSD一见钟情


## 背景

我这个人对计算机有很重的洁癖, 若系统有一点不满足我的想法, 我会很难受很厌恶.

比如win安装后若登录系统, 用户文件夹将会以你microsoft账号名命名, 在我改名后, 仍然以我旧名命名, 而且是无法修改的. 于是我直接重装+创建本地账户后再登陆.

再有, 使用着中文win后发现, 即使我改成UTF-8的编码, 使用英文系统, 仍然有大量中文, GB存在于系统, 于是我又重装, 直接上英文镜像.

再比如linux, 我需要btrfs用于增量快照, 方便我随时恢复被弄糟的系统. 结果为此重装无数次.

linux各种配置非常杂乱, 到处整, 像蟑螂下蛋一样到处下.

linux桌面很让我难受, xwindows我是真的不想碰, wayland又完全无法使用(不用兼容层).

## FreeBSD

首先感谢公司, 给我发了一台iMac电脑, 这是我第一台unix系的电脑(若MacOS也算的话). 在搜索过程中我发现了FreeBSD. 进而了解了他们的故事.

目前我对FreeBSD完全没有了解, 只是简单安装在了Hyper-V上, 也就是简单配置了下. 但是系统的设计理念, 系统的哲学非常让我愉悦, 这才是K.I.S.S, 这才是正在的小而美.

我从未如此得想知道某个事物的深层次原理, FreeBSD是第一个.

我决定用我课余时间研究它, 我的人生目标之一是成为FreeBSD的开发者之一.
