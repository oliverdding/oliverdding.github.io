# 来深圳的第一天


> 总结全文: 深圳不适合我个人发展

正午的飞机, 但是昨晚熬夜打游戏+吃了一份中华炸鸡, 一早起来喉咙非常干燥难忍.

飞机上明显感觉到自己发热了, 全程纠结着告诉空姐, 但是最后决定先不说, 因为我几乎肯定不是新冠, 为了避免恐慌, 下飞机后就做核算检测, 若为阳性再说也不迟.

同学妈妈非常热情, 载着我去到腾讯分配的中转宾馆, 这个宾馆算我住过最高级的宾馆了(一晚380): 巨大半身窗, 单人沙发, 立地吊灯...还有各种非常体贴的小东西. 我算是理解为什么明星都喜欢住宾馆而不是自己买个房子住.

紧接着开始了罪恶之旅, 就近选择了南山医院, 一路上坑坑洼洼, 堵车严重, 每个路口(没有夸张)都在修不知道什么东西, 人行道看心情存在, 有的就半米不到, 城市规划完全稀巴烂.

到医院后, 医院专门划分出发热科医治发热患者, 从富丽堂皇的医院正门旁边的小路走, 大概走了10分钟, 一路上全是施工工地, 让人觉得自己是土木学生想要提桶跑路.

偌大个医院, 发热科是用板房临时搭建的, 三个医生, 7个护士(大概)...我核算检测(GIAO! 插鼻孔, 体验极差)+等待报告大概4个小时.

还好还好, **上呼吸道感染**.

回去的路上完全找不到共享单车, 快到了突然发现扎堆的单车...

入职计划被打乱, 又请假了2, 3天, 欸=0=

总之, 这里是打工人扎堆的地方, 这里最不缺打工人, 除此之外, 什么都缺.

我看不到这个城市的生活气息, 看不到适合居住的环境, 加上这恐怖的房价, 得勒...OVER
